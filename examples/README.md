# Examples

This directory contains reference artifacts generated by the AI Workflow framework
during real project use. They show what the pipeline produces and how to use the
work-item, ADR, and planning systems effectively.

## Contents

| Path | What it shows |
|------|---------------|
| `adr/` | Sample Architecture Decision Records generated by the `adr-generator` skill |
| `goal-file-example.md` | A complete Goal File showing how to define a project vision for the pipeline |

## How to use these

These are **examples only** — they are not part of the framework itself.
Copy and adapt them as starting points for your own project.

The pipeline will generate its own ADRs in `docs/adr/` and enhancement plans
in `docs/enhancements/` as you work.
